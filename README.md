
## Basic Installation
1. Install Docker. 
2. Install Docker compose.
3. Install nvm.
4. Install PM2. 

## Deployment
1. Pull the project from repo
```bash
$ npm install
```

## Running the app

With these params.

```bah
DATABASE_USERNAME='postgres'
DATABASE_PASSWORD='postgres'
DATABASE_NAME='coastalvit'
```
```bash
# development
$ npm run start

```
